# Tabs

在一个 app 中，Tabs 使在不同的视图或功能方面探索和切换起来变得简单，或者浏览不同类别的数据集合变得简单。

## 用法

一个 tab 为显示内容相关的分组提供启示。一个 tab 标签简要地描述了相关分组的内容。

### 何时使用

使用 tabs 将大量的关联数据或选择划分地更容易理解，专注在有助于内容导航或内容
结构为目标的组，而不需要导航出当前的上下文。
